# OAK4D Pro Indoor Navigation Audit (OCR + YOLO-World)

This folder contains an **OAK4D Pro** camera application to **audit and support the design of indoor wayfinding**:

- Open-vocabulary wayfinding detection (YOLO-World) on RVC4
- OCR (EasyOCR)
- Arrow direction detection (OpenCV)
- Optional Gemini text-only route evaluation

## Run

```bash
python3 ocr.py
```
